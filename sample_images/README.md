# Sample Images for Testing

This folder should contain sample images for testing the action recognition system.

## Recommended Sample Images

Add JPG/JPEG images showing these actions:
- Playing guitar
- Reading a book
- Riding a bike
- Cooking
- Using a computer
- Drinking
- And any of the other 35 supported actions

## Usage

1. Add your sample images to this folder
2. Use the frontend interface to upload and test these images
3. Verify that the model correctly recognizes the actions

## Image Requirements

- **Format:** JPG/JPEG only
- **Size:** Maximum 10MB
- **Content:** Clear, well-lit images showing the action
- **Resolution:** Recommended 224x224 or higher

## For Assignment Submission

Include at least 5-10 sample images demonstrating different actions for evaluation purposes.
